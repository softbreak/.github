## STAR Program

Softbreak STAR, yazılım geliştiricilerin gerçek üretim süreçlerini deneyimlediği eğitim programıdır.

- GitHub tabanlı sprint döngüsü
- Issue → Branch → PR → Review → Merge akışı
- Mentor eşliğinde takım üretimi
- Mezuniyet projesi

**Başlangıç Paketi** — 4 hafta  
**Geliştirici Paketi** — 12 hafta

---

## Yaklaşım

Softbreak'te öğrenmek üretmektir.

Katılımcılar gerçek bir yazılım ekibinin çalışma ritmine girer.  
Kod yazar, review alır, hata düzeltir, teslim eder.

Amaç bilgi aktarmak değil — üretim kası kazandırmaktır.
